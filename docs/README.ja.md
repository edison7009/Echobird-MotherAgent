## 🤖 MotherAgent — モデルをデプロイ。エージェントを起動。

**MotherAgent** はあなたの自律 AI エージェントです — ローカル LLM のデプロイ、リモートモデルの接続、OpenClaw の起動をすべて一か所から行えます。

- 🖥️ **ローカル LLM デプロイ** — 内蔵 llama.cpp で Qwen、DeepSeek、Llama などをワンクリックでデプロイ。データは端末外に出ません。
- 🌐 **リモート LLM** — OpenAI、Anthropic、Google Gemini または任意の OpenAI 互換エンドポイントへ即接続。
- 🦅 **OpenClaw デプロイ** — MotherAgent から直接 OpenClaw エージェントを起動・管理。ターミナル不要。
- 💾 **永続セッション** — エージェントの会話はアプリ再起動後も保持。続きからすぐに再開できます。
- ⚡ **任意プロトコル** — OpenAI API と Anthropic API。エージェントごとにプロトコルを設定、設定変更ゼロで切替。

---

## ✨ Echobird — モデルを切り替え。設定ファイルではなく。

Echobird はすべての AI コーディングツールの**ビジュアルコントロールパネル**です。ポイント、クリック、切替。

- 🎯 **ワンクリック切替** — 任意のツールの AI モデルをビジュアルに設定。JSON ファイルを掘り返す必要なし。
- 🔀 **デュアルプロトコル** — OpenAI & Anthropic API。いつでも切替可能。
- 🚇 **スマートプロキシ** — VPN なしで地域制限 API にアクセス。
- 🧩 **スキルブラウザ** — ツールをまたいで AI スキルを発見・インストール。
- 🎮 **内蔵 AI アプリ** — Reversi、AI 翻訳など。続々追加予定。
- 🌍 **28 言語** — グローバル開発者向けの完全国際化対応。
