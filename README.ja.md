## 🤖 MotherAgent — 主要機能

**MotherAgent** は Echobird に内蔵された自律型 AI エージェントです：

- 🧠 **思考と計画** — 複雑なタスクを実行可能なステップに分解
- 🔧 **ツールの使用** — Shell コマンドの実行、ファイルの読み取り、Web ブラウジング
- 🔄 **自己修正** — 出力を確認し、エラー時に再試行
- 💾 **永続メモリ** — アプリ再起動後も会話履歴を保持
- ⚡ **任意の LLM に対応** — OpenAI、Anthropic、またはローカルモデルに接続

---

## ✨ Echobird とは？

Echobird は、AI コーディングツール全体で**統一されたモデル管理インターフェース**を提供するデスクトップアプリです。

- 🎯 **ワンクリックモデル切替** — 任意のツールの AI モデルを視覚的に切替
- 🔀 **デュアルプロトコル** — OpenAI & Anthropic API 対応
- 🚇 **スマートトンネルプロキシ** — VPN なしで制限された API にアクセス
- 🧩 **スキルブラウザ** — AI スキルの発見・インストール・管理
- 🖥️ **ローカル LLM サーバー** — llama.cpp でオープンソースモデルをローカル実行
- 🌍 **28 言語対応** — 完全な国際化サポート
- 🌃 **サイバーパンク UI** — ネオングリーンのターミナル美学
